Firebase 동기화 (PC·폰 데이터 공유)
미연결 · 한 번만 설정하면 기기 간 자동 동기화됩니다.
📖 처음 설정 가이드 (5분)
1. console.firebase.google.com 접속 → 프로젝트 만들기 (무료 Spark 요금제)
2. 빌드 → Authentication → 시작하기 → 로그인 방법에서 이메일/비밀번호 사용 설정
3. 빌드 → Firestore Database → 데이터베이스 만들기 (프로덕션 모드, 위치 아무거나)
4. Firestore의 규칙 탭에 아래를 붙여넣고 게시:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /budget/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } } } 5. 프로젝트 개요 옆 ⚙️ → 프로젝트 설정 → 하단 "내 앱" → 웹 앱(</>) 추가 → firebaseConfig 코드를 복사해 아래 칸에 붙여넣기
6. 이메일/비밀번호를 정해 회원가입 → 끝! 다른 기기에서는 같은 설정+로그인만 하면 돼요.
